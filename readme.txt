use this link to hit the api --- https://task-scheduler-gules.vercel.app/
send data on the request body 
/api/auth/register - for registering
/api/auth/login - for login
/api/tasks - for creating task
/api/tasks/:id/complete - for completing task
/task - for retrieving task and filtering  use options like priority,status,sortby in body to sort accodingly
